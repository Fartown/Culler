## 问题与目标
- 问题：单图预览缺少右键菜单与删除键快捷操作，导致无法快速执行常用功能。
- 目标：在不引入大范围重构的前提下，尽快为单图预览模式提供与网格视图一致的右键菜单，并使 Delete 键生效。

## 解决思路（最小改动）
- 添加右键菜单：在 SinglePhotoView 的图片容器挂载 PhotoContextMenu(targets: [photo])。
- 增加删除键分发：在 KeyboardShortcutManager 监听 Delete（keyCode=51），通过通知分发给当前视图。
- 本地实现删除动作：为避免跨文件重构，直接在 SinglePhotoView 内实现三类删除（复用现有逻辑思路，代码最小复制）。

## 具体改动
1. SinglePhotoView 增加右键菜单
   - 在显示图片的 ZStack/Image 节点添加：
     ```swift
     .contextMenu { PhotoContextMenu(targets: [photo]) }
     ```
   - 参考：网格视图挂载位置 [PhotoGridView.swift:L52-L58](file:///Users/zhangchao/workspace/dev/Culler/Culler/Culler/Views/PhotoGridView.swift#L52-L58)。
2. KeyboardShortcutManager 监听 Delete 键
   - 在 handle(_:) 中加入：
     ```swift
     if event.keyCode == 51 {
         NotificationCenter.default.post(name: .deletePhoto, object: event.modifierFlags)
         return nil
     }
     ```
   - 保持现有“文本焦点/弹窗绕过”判断不变。[KeyboardShortcutManager.swift:L34-L71](file:///Users/zhangchao/workspace/dev/Culler/Culler/Culler/Services/KeyboardShortcutManager.swift#L34-L71)
3. SinglePhotoView 订阅删除通知并执行删除
   - 添加 @Environment(\.modelContext) 以更新数据模型。
   - 订阅通知：
     ```swift
     .onReceive(NotificationCenter.default.publisher(for: .deletePhoto)) { note in
         let mods = note.object as? NSEvent.ModifierFlags ?? []
         if mods.contains(.option) {
             deleteRawJpgPair(photo)
         } else if mods.contains(.shift) {
             deleteFromDisk(photo)
         } else {
             removeFromLibrary(photo)
         }
     }
     ```
   - 删除完成后：若仍有上一/下一张，自动导航，避免空白视图。
4. 在 SinglePhotoView 内部实现删除动作（最小复制）
   - removeFromLibrary(photo): `modelContext.delete(photo)`。
   - deleteFromDisk(photo): 依据书签访问并使用 FileManager 删除当前文件，随后从库移除。
   - deleteRawJpgPair(photo): 计算 RAW/JPG 配对路径并删除；如库中存在对应 Photo，一并从库移除。
   - 参考实现与调用方式：[PhotoContextMenu 删除相关](file:///Users/zhangchao/workspace/dev/Culler/Culler/Culler/Views/PhotoGridView.swift#L431-L471)、[RAW+JPG 配对删除](file:///Users/zhangchao/workspace/dev/Culler/Culler/Culler/Views/PhotoGridView.swift#L473-L546)。

## 快捷键约定
- Delete：从库移除当前照片（安全默认）。
- Shift+Delete：从磁盘删除当前文件。
- Option+Delete：从磁盘删除 RAW+JPG 配对。

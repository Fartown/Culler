## KV隐藏逻辑
- 无值（nil 或空字符串）不渲染该 KV 行；为非可选值保留
  - 相机/曝光：已有 if let，保留；新增空白判定扩展以覆盖字符串
  - 标签（tags）：当前为“无”，按需求改为无值则不显示该行
  - 文件大小：为明确值（可能为 0），保留显示
- 整组隐藏：当组内所有 KV 都不存在时，不渲染该 DisclosureGroup
  - hasCameraInfo = make/model/lens 任一存在
  - hasExposureInfo = focalLength/aperture/shutter/iso 任一存在
  - hasMarkingInfo = flag≠none 或 rating>0 或 colorLabel≠none 或存在 tags
  - hasFileInfo = 宽高/文件路径/大小任一存在（通常存在）

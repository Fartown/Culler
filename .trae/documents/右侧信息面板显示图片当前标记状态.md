## 目标
- 当图片被选中时，右侧信息面板展示该图片的当前标记状态：Flag（Pick/Reject/None）、Rating（星级）、Color Label（颜色标签）以及 Tags（标签列表）。

- 右侧信息面板视图在单张或网格模式下由 ContentView 传入当前选中 Photo 并渲染：[ContentView.swift:L118-L122](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/ContentView.swift#L118-L122)。
- 信息面板当前展示缩略图、基本信息、Camera/Exposure/File 三个分组：[InfoPanelView.swift](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/InfoPanelView.swift)。
- 标记相关模型字段位于 Photo：rating、flag/colorLabel 以及关联 tags：[Photo.swift:L51-L78](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Models/Photo.swift#L51-L78)。
- 现有标记操作入口在底部工具栏（批量）：[MarkingToolbar.swift](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/MarkingToolbar.swift)，标签管理 UI（TagChip/FlowLayout）可复用：[AlbumManagementView.swift:L205-L226](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/AlbumManagementView.swift#L205-L226)。

## 实现方案
1. 在 InfoPanelView 中新增一个 DisclosureGroup“Marking”（默认展开），用于集中展示当前图片的标记状态。
2. Flag 行：根据 photo.flag 显示对应图标与文本（Pick/Reject/None）。
3. Rating 行：以星星图标展示 photo.rating 数量，并附数字说明（如 3/5）。
4. Color Label 行：以彩色圆点（使用 Color(photo.colorLabel.color)）与名称（Red/Yellow/…）展示。
5. Tags 行：若 photo.tags 非空，使用 FlowLayout + TagChip 列表展示；为空时显示“No tags”。
6. 保持现有 InfoPanelView 的配色、间距与分隔线风格一致；新增 @State private var expandMarking: Bool = true 控制展开。
7. 不引入交互修改（只读展示），后续如需在面板直接修改，可复用现有 Flag/Rating/ColorLabel 组件并加以禁用或精简。

## 技术细节
- 数据来源：Photo 为 @Model 引用类型，标记变更（来自底部工具栏或右键菜单）会自动刷新绑定视图，无需额外通知。
- 复用组件：直接引用 TagChip 与 FlowLayout 来渲染标签列表：[AlbumManagementView.swift:L205-L226](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/AlbumManagementView.swift#L205-L226)。
- 代码位置：在现有 InfoPanelView 的三组（Camera/Exposure/File）之前或之后插入“Marking”分组，遵循同样的 DisclosureGroup/Divider 结构：[InfoPanelView.swift](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/InfoPanelView.swift)。

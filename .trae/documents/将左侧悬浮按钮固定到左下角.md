**目标**

* 左右两个悬浮按钮统一到顶部，分别位于左上角与右上角，位置一致。

**代码位置**

* 左侧按钮叠加：[ContentView.swift:L180-L191](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/ContentView.swift#L180-L191)

* 右侧按钮叠加：[ContentView.swift:L193-L204](file:///Users/bytedance/workspace/ai-demo/culler/Culler/Culler/Views/ContentView.swift#L193-L204)

**具体修改**

* 将左侧 `.overlay(alignment: .leading)` 改为 `.topLeading`，右侧 `.overlay(alignment: .trailing)` 改为 `.topTrailing`；并增加顶部与边缘间距保持美观与一致。

```swift
// 左侧
.overlay(alignment: .topLeading) {
    if !showLeftNav {
        Button {
            withAnimation(.spring(response: 0.35, dampingFraction: 0.85, blendDuration: 0.2)) { showLeftNav = true }
        } label: {
            Image(systemName: "sidebar.leading")
                .padding(6)
        }
        .buttonStyle(.plain)
        .background(Color(NSColor(hex: "#2a2a2a")))
        .cornerRadius(6)
        .padding(.leading, 8)
        .padding(.top, 8)
    }
}

// 右侧
.overlay(alignment: .topTrailing) {
    if !showRightPanel {
        Button {
            withAnimation(.spring(response: 0.35, dampingFraction: 0.85, blendDuration: 0.2)) { showRightPanel = true }
        } label: {
            Image(systemName: "sidebar.trailing")
                .padding(6)
        }
        .buttonStyle(.plain)
        .background(Color(NSColor(hex: "#2a2a2a")))
        .cornerRadius(6)
        .padding(.trailing, 8)
        .padding(.top, 8)
    }
}
```

**原因说明**

* `.leading`/`.trailing` 默认是垂直居中；改为 `.topLeading`/`.topTrailing` 可统一放到顶部，确保左右一致。

* 适当 `.padding` 保持与窗口边缘的距离，避免紧贴。

**验证**

* 隐藏两侧面板时，左右按钮均出现在顶部两侧；点击后能分别唤起对应面板。

* 在网格/单图/文件夹浏览模式下位置稳定且不与顶部工具条视觉冲突。

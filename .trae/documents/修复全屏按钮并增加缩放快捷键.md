## 修改方案
### 1) 修复单张视图的全屏按钮
- 在单张视图右上角按钮点击时发送进入全屏通知：Notification.Name.enterFullscreen
- 在 ContentView 订阅该通知，将 viewMode 切换为 .fullscreen（保持现有过渡动画）

### 2) 增加缩放快捷键
- 在 KeyboardShortcutManager 中拦截：
  - Command + =（以及 Command + Shift + = 以兼容“+”）→ .zoomIn
  - Command + - → .zoomOut
  - Command + 0 → .zoomReset
- 通过 NotificationCenter 分发上述通知（新增：zoomIn/zoomOut/zoomReset）

### 3) 单张视图响应缩放通知
- SinglePhotoView 订阅 .zoomIn/.zoomOut/.zoomReset：
  - zoomIn：scale = min(scale * 1.2, 5.0)
  - zoomOut：scale = max(scale / 1.2, 1.0)
  - zoomReset：scale = 1.0，offset = .zero
- 当缩放值回到 1.0 时，offset 自动清零，保持与现有双击行为一致

### 4) 可选：全屏视图同步支持上述缩放快捷键
- 如需一致体验，可在 FullscreenView 增加 scale 状态与同样的通知处理（不改动现有退出/导航逻辑）

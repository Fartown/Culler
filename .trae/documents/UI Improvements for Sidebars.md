# 侧边栏 UI 优化与布局调整

## 右侧侧边栏 (检查器) - `InfoPanelView.swift`
1.  **KV 横向排列**:
    *   将垂直排列的 `InfoRow` (标签在上，值在下) 改为横向排列。
    *   使用 `Grid` 和 `GridRow` 布局，确保标签和值在不同行之间对齐，视觉效果更整洁优雅。
    *   布局样式：`标签 (右对齐/次要颜色) | 值 (左对齐/主要颜色)`。

2.  **隐藏空数据**:
    *   **隐藏无值的 KV**: 确保如果值为 nil 或空字符串，则不渲染该行。
    *   **隐藏空组**: 实现计算属性 (如 `hasCameraInfo`, `hasExposureInfo`) 来检查各部分是否有数据。如果某一组内所有字段都为空，则隐藏整个折叠组 (`DisclosureGroup`)。

3.  **优化“标记”部分**:
    *   将旗标、评分、颜色标签部分也调整为匹配新的横向 Grid 布局，保持整体统一。

## 左侧侧边栏 - `SidebarView.swift`
1.  **左对齐修复**:
    *   顶部的“导入照片”按钮目前是居中的。我将其调整为靠左对齐，与列表内容保持一致。
    *   检查并强制所有列表行和自定义视图使用 `.leading` (靠左) 对齐。

2.  **视觉优化**:
    *   调整侧边栏头部的间距和填充，使其与 UI 其余部分保持协调。

## 修改文件
- `InfoPanelView.swift`: 重写 `InfoPanelContent`，使用 `Grid` 展示元数据，实现隐藏空组/行的逻辑。
- `SidebarView.swift`: 调整头部按钮对齐方式，确保整体左对齐。
